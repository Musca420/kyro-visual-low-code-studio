# Action map

Component events include click, double click, long press, pointer enter/leave, submit, input, change, focus, blur, key down, scroll and swipe left/right. Page events include load, visible, hidden, online, offline, timer and device shake.

Node categories:

- UI: navigate, modal, update UI, notification, refresh;
- logic: validate, condition, switch, delay, debounce, loop;
- state and data: get/set/reset, query, insert, update, delete, filter, sort, KPI;
- services: HTTP and file;
- device: request permission, platform condition, native action;
- advanced: typed extension module and debug log.

An event link belongs to the component. A flow remains reusable after that link is disconnected.
