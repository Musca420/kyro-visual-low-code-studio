# Publish operations

- `set_export_config {patch}` selects web, PWA or Android and target settings.
- `export_project {format}` requests project JSON or runnable app export.
- `prepare_android {build}` creates/synchronizes the Capacitor Android project and optionally builds an APK.
- Folder import remains a user-selected filesystem action; inspect the imported graph afterward.

Verify generated files are readable, install dependencies, build, launch separately, and exercise the same acceptance path. Never claim public Windows trust without a publicly trusted signing certificate.
